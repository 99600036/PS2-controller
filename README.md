# PS2 Controller

Sample joystick positions with STM32F072B and send over serial to
[STM-Quadcopter](http://github.com/zegervdv/STM-Quadcopter)

## Wiring
| PS2 colour | Button |
| ---------- | ------ |
| Yellow     | RX     |
| Blue       | RY     |
| Brown      | RZ     |
| Green      | LX     |
| Purple     | LY     |
| Orange     | LZ     |
